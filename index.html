01 · Teori & Bukti Formal

Kebebasan Linear

Definisi, teorema, metode pengujian, dan hubungannya dengan basis serta dimensi ruang vektor.

Definisi Formal

Himpunan vektor {v₁, v₂, …, vₖ} dalam ruang vektor V disebut bebas linear jika satu-satunya solusi untuk persamaan berikut adalah solusi trivial:

c₁·v₁ + c₂·v₂ + · · · + cₖ·vₖ = 0 ⟹ c₁ = c₂ = · · · = cₖ = 0

Jika terdapat skalar cᵢ ≠ 0 yang memenuhi persamaan di atas, maka himpunan tersebut bergantung linear (linearly dependent) — artinya minimal satu vektor dapat dinyatakan sebagai kombinasi linear vektor-vektor lain dalam himpunan.

✓ Bebas Linear
Tidak ada vektor yang merupakan kombinasi linear dari vektor lainnya. Setiap vektor menambah "dimensi baru".
v₁=(1,0), v₂=(0,1) det = 1·1 − 0·0 = 1 ≠ 0 ✓
✗ Bergantung Linear
Ada vektor yang "redundan" — bisa dibentuk dari kombinasi vektor lain dalam himpunan.
v₁=(1,2), v₂=(2,4) v₂ = 2·v₁ → bergantung!
Intuisi Geometris
ℝ²

Dua vektor bebas linear ⟺ tidak kolinear. Mereka menunjuk ke arah yang berbeda dan bisa membentuk seluruh bidang ℝ².

ℝ³

Tiga vektor bebas linear ⟺ tidak koplanar. Mereka tidak semuanya berbaring di bidang yang sama dan bisa mengisi seluruh ruang ℝ³.

ℝⁿ

Maksimal n vektor bebas linear dapat ada di ℝⁿ. Lebih dari n vektor di ℝⁿ pasti bergantung linear.

1

Metode Determinan — hanya untuk n vektor di ℝⁿ (matriks persegi)

Susun vektor sebagai kolom → A = [v₁|v₂|···|vₙ] det(A) ≠ 0 ⟺ Bebas Linear det(A) = 0 ⟺ Bergantung Linear
2

Eliminasi Gauss (Reduksi Baris) — berlaku umum untuk m×k

Buat A = [v₁|v₂|···|vₖ], lakukan REF: • Hitung rank(A) • Bebas linear ⟺ rank(A) = k (semua kolom pivot) • Bergantung ⟺ rank(A) < k (ada baris nol)
3

Wronskian — untuk fungsi-fungsi di ruang fungsi

W(f₁,…,fₙ)(t) = det([[f₁,…,fₙ],[f₁',…,fₙ'],[⋮,⋮,⋮],[f₁⁽ⁿ⁻¹⁾,…,fₙ⁽ⁿ⁻¹⁾]]) W(t₀) ≠ 0 ⟹ Bebas Linear
Teorema 1 — Vektor Nol

Himpunan yang memuat vektor nol selalu bergantung linear.

Teorema 2 — Kelebihan Vektor

Jika k > n, maka k vektor di ℝⁿ pasti bergantung linear.

Teorema 3 — Karakterisasi

{v₁,…,vₖ} bergantung linear ⟺ ∃ vⱼ yang merupakan kombinasi linear dari v₁,…,vⱼ₋₁.

Teorema 4 — Rank-Nullity

Untuk matriks A ukuran m×n: rank(A) + nullity(A) = n

Definisi Basis

Himpunan B = {v₁,…,vₙ} adalah basis untuk V jika: (1) B bebas linear, dan (2) span(B) = V.

dim(V) = n ⟺ Setiap basis V memiliki tepat n vektor Bebas linear + n vektor di ℝⁿ ⟺ Basis ℝⁿ rank(A) = n ⟺ Kolom-kolom A membentuk basis Col(A)
Basis Standar ℝ³
e₁=(1,0,0) e₂=(0,1,0) e₃=(0,0,1) det = 1 ≠ 0 ✓
Basis Polinomial P₂
B = {1, t, t²} dim(P₂) = 3 span = p(t), deg ≤ 2
Implikasi Penting

Kolom-kolom A bebas linear ⟺ Null(A) = {0} ⟺ nullity(A) = 0 ⟺ rank(A) = n (untuk A ukuran m×n).

Contoh: Metode Determinan
Uji kebebasan linear menggunakan det(A)
Soal

Apakah v₁=(1,2), v₂=(3,4) bebas linear di ℝ²?

1

Susun sebagai matriks:

A = [v₁|v₂] = |1 3| |2 4|
2

Hitung determinan:

det(A) = (1)(4) − (3)(2) = 4 − 6 = −2
3

Interpretasi:

det(A) = −2 ≠ 0, maka v₁ dan v₂ bebas linear. ✓

Contoh Bergantung

Untuk v₁=(1,2), v₂=(2,4): det = 1×4 − 2×2 = 0. Bergantung! (v₂ = 2v₁)

Contoh: Eliminasi Gauss
Row Reduction untuk mencari rank
Soal

Apakah v₁=(1,2,3), v₂=(0,1,2), v₃=(0,0,1) bebas linear di ℝ³?

1

Bentuk matriks:

A = |1 0 0| |2 1 0| |3 2 1|
2

Eliminasi → REF:

R₂ ← R₂ − 2R₁: |0 1 0| R₃ ← R₃ − 3R₁: |0 2 1| R₃ ← R₃ − 2R₂: |0 0 1| Hasil: |1 0 0| |0 1 0| |0 0 1| = I₃
3

Hitung rank: rank(A) = 3 = jumlah vektor → bebas linear. ✓

Bukti Teorema 1
Himpunan berisi vektor nol selalu bergantung linear
Pernyataan

Jika S = {0, v₁, …, vₖ₋₁}, maka S bergantung linear.

1

Pertimbangkan kombinasi linear: c·0 + 0·v₁ + … + 0·vₖ₋₁ = 0

2

Kombinasi ini memiliki koefisien c ≠ 0 (ambil sembarang c ≠ 0, misalnya c = 1).

3

Karena terdapat solusi non-trivial untuk Σcᵢvᵢ = 0, maka S bergantung linear menurut definisi. □

Bukti Teorema 2
k > n vektor di ℝⁿ selalu bergantung linear
Pernyataan

Jika k > n, maka {v₁,…,vₖ} ⊂ ℝⁿ pasti bergantung linear.

1

Susun vektor sebagai kolom matriks A berukuran n × k.

2

Pertimbangkan sistem homogen Ac = 0 di mana c ∈ ℝᵏ. Sistem ini memiliki n persamaan dan k variabel.

3

Karena k > n, jumlah variabel melebihi jumlah persamaan. Oleh Teorema Fundamental Aljabar Linear, sistem ini pasti memiliki solusi non-trivial (setidaknya satu variabel bebas).

4

Adanya c ≠ 0 dengan Ac = 0 berarti c₁v₁ + … + cₖvₖ = 0 dengan tidak semua cᵢ = 0. Maka himpunan bergantung linear. □

Implikasi dari Rank-Nullity

rank(A) ≤ n < k, sehingga nullity(A) = k − rank(A) ≥ k − n ≥ 1. Ada solusi non-trivial.

Bukti Teorema 3
Karakterisasi bergantung linear
Pernyataan (Biimplikasi)

{v₁,…,vₖ} bergantung linear ⟺ ∃ j: vⱼ = kombinasi linear dari {v₁,…,vⱼ₋₁}

(⟹) Arah pertama:

1

Jika bergantung linear, ∃ c ≠ 0 dengan Σᵢcᵢvᵢ = 0.

2

Ambil j = indeks terbesar dengan cⱼ ≠ 0.

3

Maka: cⱼvⱼ = −Σᵢ﹤ⱼ cᵢvᵢ

4

Karena cⱼ ≠ 0, bagi kedua sisi: vⱼ = −(1/cⱼ)Σᵢ﹤ⱼ cᵢvᵢ. ✓

(⟸) Arah balik:

1

Jika vⱼ = Σᵢ﹤ⱼ aᵢvᵢ, susun ulang: a₁v₁ + … + aⱼ₋₁vⱼ₋₁ − 1·vⱼ = 0.

2

Koefisien vⱼ = −1 ≠ 0, jadi ini solusi non-trivial. Maka bergantung linear. □

Teorema Rank-Nullity
Untuk matriks A berukuran m × n
rank(A) + nullity(A) = n
Ide Bukti

Misalkan rank(A) = r. Setelah eliminasi Gauss, terdapat r kolom pivot dan (n−r) variabel bebas.

1

Null(A) = {x ∈ ℝⁿ : Ax = 0}. Setiap solusi ditulis dalam (n−r) parameter bebas.

2

Setiap variabel bebas memberikan satu vektor basis untuk Null(A).

3

Maka nullity(A) = n − r = n − rank(A), sehingga rank + nullity = r + (n−r) = n. □

Implikasi Utama

Kolom A bebas linear ⟺ nullity(A) = 0 ⟺ rank(A) = n ⟺ Ax=0 hanya punya solusi trivial.

Kompresi Gambar — SVD
Singular Value Decomposition & Linear Independence

Setiap gambar digital bisa direpresentasikan sebagai matriks intensitas pixel. SVD memfaktorkan matriks ini menjadi komponen-komponen yang saling bebas linear.

A_{m×n} = U_{m×r} · Σ_{r×r} · Vᵀ_{r×n} Kolom U: vektor singular kiri (bebas linear, ortonormal) Kolom V: vektor singular kanan (bebas linear, ortonormal) Σ: nilai singular σ₁ ≥ σ₂ ≥ · · · ≥ σᵣ > 0
Mengapa bebas linear penting?

Kolom U dan V yang bebas linear menjamin tidak ada informasi yang tumpang tindih. Rank-k approximation (membuang σ kecil) memberikan kompresi optimal karena setiap komponen membawa informasi unik.

Machine Learning — PCA
Principal Component Analysis

PCA mencari arah-arah utama variasi dalam data. Arah-arah ini (principal components) harus bebas linear agar tidak ada informasi redundan.

Data X ∈ ℝ^{n×p} (n sampel, p fitur) C = (1/n)XᵀX ← matriks kovarians Eigenvectors {v₁,…,vₖ} dari C: bebas linear PC₁ = Xv₁, PC₂ = Xv₂, … PCᵢ ⊥ PCⱼ untuk i ≠ j (ortogonal ⟹ bebas linear)
Aplikasi Nyata

Pengenalan wajah (Eigenfaces), rekomendasi konten, visualisasi data dimensi tinggi — semuanya mengandalkan principal components yang bebas linear.

Pemrosesan Sinyal — FFT
Fourier Transform & Basis Ortogonal

Transformasi Fourier menyatakan sinyal apapun sebagai kombinasi linear dari fungsi sin dan cos. Fungsi-fungsi ini terbukti bebas linear melalui ortogonalitas.

⟨sin(mt), sin(nt)⟩ = ∫₀²π sin(mt)sin(nt)dt = π·δₘₙ ⟨cos(mt), cos(nt)⟩ = ∫₀²π cos(mt)cos(nt)dt = π·δₘₙ ⟨sin(mt), cos(nt)⟩ = 0 untuk semua m,n Basis Fourier = {1, cos(t), sin(t), cos(2t), sin(2t), …} Semua elemen BEBAS LINEAR → rekonstruksi sinyal unik
Mengapa penting?

Bebas linear menjamin setiap frekuensi membawa informasi unik — dasar kompresi MP3, transmisi WiFi/4G/5G, dan MRI di dunia medis.

QR Code & Kriptografi
Reed-Solomon & Kode Linear

Error-correcting codes (QR Code, CD, DNS) menggunakan ruang vektor dengan basis bebas linear untuk mendeteksi dan memperbaiki error.

Kode linear C ⊂ 𝔽₂ⁿ G = generator matrix (k × n) Baris G bebas linear → 2ᵏ codeword unik Reed-Solomon di GF(2⁸): → Koreksi hingga 30% error (QR Level H) → Baris G bebas linear di atas GF(256)
Contoh nyata

Saat HP scan QR code yang lecet/kotor, Reed-Solomon memperbaiki error karena basis bebas linear menjamin setiap codeword dapat direkonstruksi secara unik.

Rangkaian Listrik
Hukum Kirchhoff & Sistem Linear
Analisis 3 mesh: [R₁₁ R₁₂ R₁₃] [I₁] [V₁] [R₂₁ R₂₂ R₂₃] [I₂] = [V₂] [R₃₁ R₃₂ R₃₃] [I₃] [V₃] det(R) ≠ 0 ⟺ baris bebas linear ⟺ solusi arus unik ada
Kapan gagal?

Jika dua persamaan mesh bergantung linear (redundan), arus tidak bisa ditentukan secara unik. Ini terjadi pada rangkaian yang under-constrained atau memiliki loop ekstra yang tidak independen.

Grafis 3D & Game Engine
Transformasi Koordinat & Gimbal Lock
Basis standar ℝ³: {i,j,k} bebas linear Rotasi X: Rₓ(θ) = [[1,0,0],[0,cosθ,-sinθ],[0,sinθ,cosθ]] det(Rₓ) = 1 ≠ 0 → bebas linear → invertible Proyeksi perspektif (homogeneous coords): P ∈ ℝ^{4×4}, det(P) ≠ 0
Gimbal Lock — Masalah Terkenal

Saat dua sumbu rotasi menjadi kolinear (bergantung linear), satu derajat kebebasan hilang. Solusi: quaternion (4D), yang menjaga representasi bebas dari degenerasi linear.
